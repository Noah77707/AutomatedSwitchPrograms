Built using code from Dinones program called Nintendo-Switch-Pokemon-Shiny-Hunter, esp32beans program called switch_ESP32, along with some debuging code from github.

Buttons:
A = 0
B = 1
X = 2
Y = 3
LeftTrigger = 4
RightTrigger = 5
LeftThrottle = 6
RightThrottle = 7
Minus = 8
Plus = 9
LeftStick = 10
RightStick = 11
Home = 12
Capture = 13
Reservered1 = 14
Reservered2 = 15
